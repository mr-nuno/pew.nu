About

Basics

This is a nerd site for us playing EA Sports NHL Franchise.

This site consists of four parts.
  1. Statistics - Where you check yours and others current stats.
  2. Recent games - Where you check your recent games.
  3. How hot - How hot are you, based on your recent games.
  4. Leaderboard - Who is number 1?
  5. Settings - A setup page for simplicity.
    1. Name - [optional]
    2. Email - [optional]
    3. Gamertag - [required] Required in the sense that you get all zeros from EA if you enter an invald gamertag
    4. Game - [required]
    5. Platform - [required]
    6. Number of recent games - [optional] How many games would you want listed under recent games? defaults to 15.
    7. Other gamers - [optional] Some friends/enemies that you want to keep track off. The stats will be listed side-by-side under Statistics.
Tech

Build on Backbone, RequireJS, Handlebars, ASP.NET MVC 4
